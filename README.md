# KhunSuek DSPS Shirt Store

เวอร์ชัน Mobile-first สำหรับขายเสื้อ DSPS Cheer Jersey และชุดโปรโมชั่นพี่ขุนช่วยใคร

## อัปเดตล่าสุด
- เปลี่ยนระบบ JavaScript เป็นไฟล์เดียวแบบ classic script เพื่อให้ปุ่มเพิ่มตะกร้าทำงานทั้งการเปิดไฟล์โดยตรงและ GitHub Pages
- ใส่ข้อมูลธนาคาร ttb และโลโก้ธนาคาร
- เลขบัญชี 980-2-10437-3
- ชื่อบัญชี จิรพล ประสพสุข
- ลบ QR Code
- ลบรูปตารางไซซ์ เหลือข้อมูลไซซ์แบบข้อความและตาราง
- ปรับหน้าชำระเงินและทุกหน้าให้เน้นมือถือ

## หมายเหตุ
ระบบสั่งซื้อยังเป็น frontend demo ผ่าน LocalStorage ต้องเชื่อม Google Apps Script/Google Sheet ก่อนใช้งานจริง

- อัปเดตหน้า success: ไม่แสดงเลขคำสั่งซื้อ และใส่ข้อความขอบคุณพร้อมลิงก์ Facebook Page ที่ถูกต้อง

- ปรับโลโก้ ttb ให้เล็กและอยู่มุมบนของการ์ดธนาคาร
- รูปย่อในตะกร้าเปลี่ยนเป็นภาพนายแบบนางแบบ
- โปรโมชั่นพี่ขุนช่วยใครเป็นพรีออเดอร์และจัดส่งพร้อมเสื้อ
- เพิ่มแถบยืนยันคำสั่งซื้อแบบติดด้านล่างสำหรับมือถือ
